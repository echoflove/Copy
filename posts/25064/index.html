# 基本 / Basic

联络用名：安息 / Ansies标识号：001
传承级别：君主属性：☪ 暗属性
势力：凛冬-诺尔伦苏 / Ansies称号：不死の贵族
战斗适应：攻击 / 近距离

# 印象 / Impression

真正意义上，醒来见到的第一个人呢 —— 虽然是以不那么友善的方式。

# 战斗

# 天赋 / Talent

　 　 　 　 　 　
基础 初始状态/最大值
攻击：9933防御：9933
生命：9933会心/同调率：99/33
特殊
光の特攻Ⅲ
　对光属性敌人造成伤害增加15%。
悄然之间
　侦察阶段生效：己方总计发现5格及以上的有利地形时，对随机1个非精英敌方造成致死伤害；总计发现10格及以上的有利地形时，己方全体获得1点行动力。

# 技能 / Skill

主动
肃杀
使用间隔：3回合
对敌方单体造成4倍的暗属性伤害
  • 如果自身与敌方均处于安息的有利地形，则改为造成3次，每次2倍的暗属性伤害。
寂静之地の群风
使用间隔：7回合
对前方4*3范围内的全体敌人造成3倍的暗属性伤害。
  • 如果自身与范围内至少1个敌方处于安息的有利地形，则改为造成4次，每次1.2倍的暗属性伤害。
至高之雪の埋葬
~爆发~
对敌方单体造成8倍的暗属性伤害，并施加☪雪の因梦，持续2回合。
雪の因梦：
  • 无法动作，持续1回合
  • 无法获取爆发凝聚
  • 雪の因梦消失时，若敌方在雪の因梦生效期间行动距离为0，则对其追加使用一次肃杀
被动
随徊风流泻
每次造成伤害，对目标3*3范围内的敌人也造成0.1倍暗属性伤害。
诺尔伦苏·静默领域
当自己位于有利地形时，使己方全体获得☪凛冬の威严；并且场上每存在1格有利地形，使自己获得1层☪恩赐の绝灭。
凛冬の威严：
  • 全属性攻击力+15%
  • 全属性防御力+15%
恩赐の绝灭：
  • 每一层恩赐の绝灭使自身暗属性攻击力+1%
  • 达到10层时，每次攻击有30%概率对敌人施加脆弱：受到伤害+12%，持续2回合。

# 解放武器 / Weapon

　 　 　 　 　 　
名称：告死の黑风类型：双刀
攻击：9933防御：9933
生命：9933会心伤害：99%
技能效果
瞬の亡  正位效果
敌方单体每存在1个弱体效果，自身获得1层☪绞：每1层使自身对瞬の亡目标造成伤害+5%，持续2回合。然后对敌方单体造成5次，每次2倍的暗属性伤害。
轻送  副位效果
己方全体每次进行通常攻击时，有70%概率额外造成一次1倍暗属性伤害
刹那  副位效果
  • 暗属性角色的攻击力+15%
  • 暗属性角色的会心+12%
